# what comes next. &gt; connecting the dots..

Ein interaktiver Schulweg-Szenario-Explorer für Filippa.

Jede Entscheidung ist ein Stern; der gewählte Weg verbindet die Punkte zu einer
Linie, die auf einen Zielstern zuläuft: **NYU · Dermatologie · New York**.
Inspiriert von Steve Jobs' „You can't connect the dots looking forward; you can
only connect them looking backward." (Stanford, 2005).

## Was drin ist

Eine einzige, eigenständige Datei: [`index.html`](./index.html). Kein Build,
keine Abhängigkeiten, kein Framework — einfach im Browser öffnen.

## Wege im Szenario

- **Heute** – Gymnasium Hermannswerder, 9. Klasse wiederholt
- **Potsdam** – Da-Vinci-Gesamtschule → Abitur → Medizin (DE) oder USA
- **England** – Milton Abbey: 3 Monate / 1 Jahr / GCSE (+ A-Levels)
- **USA** – College → Medical School → Dermatology Residency → NYU

## Auf Vercel deployen

Da `index.html` im Wurzelverzeichnis liegt, braucht es **keine Konfiguration**:

1. Dieses Repo bei Vercel als neues Projekt importieren (Framework: *Other*).
2. Deploy klicken — fertig.

Jeder Push auf den Produktions-Branch deployt automatisch neu.
